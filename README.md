## Rodar a aplicação

Rode todas as aplicações com o comando:
    
```bash
docker-compose up -d
```

Este comando irá subir todos os containers necessários para rodar todo o projeto

Acesse as pastas `golang`, `django` e `nextjs` e siga as instruções.
